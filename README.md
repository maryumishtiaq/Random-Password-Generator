# Random Password Generator

## Project Overview
This project is a simple Random Password Generator developed in Python as part of DecodeLabs Industrial Training Project 3. It generates secure random passwords based on the length entered by the user.

## Features
- Generate random passwords
- User chooses password length
- Uses letters and numbers
- Simple console-based application

## Technologies Used
- Python 3
- random module
- string module

## How to Run

1. Open the project in PyCharm or VS Code.
2. Run the file:
   ```
   password_generator.py
   ```
3. Enter the desired password length.
4. The program will generate a random password.

## Sample Output

```
Enter Password Length: 8

Generated Password: A9kP3mX2
```
